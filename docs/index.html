Featured Projects

1. Interactive Movie Database

Built a responsive web app using HTML, CSS Grid, and vanilla JavaScript that allows users to search, filter, and sort movie titles from a public API. Implemented pagination, dynamic search suggestions, and mobile-first design.

2. Somatic Movement Demonstration

Created a movement study video illustrating Bartenieff Fundamentals, focusing on Core–Distal and Upper–Lower connectivity. Presented at the Allegheny College movement showcase with annotated analysis.

3. Soundscape Podcast Series

Produced a three-episode audio series that tells a short narrative purely through environmental sound and music. Handled all field recordings, foley work, and final mix in Audacity.
